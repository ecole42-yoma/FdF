# FdF

* This project is about creating a simplified 3D graphic representation of a relief landscape.
* FdF offers to represent “iron wire” meshing in 3D.

<br>
  
# Examples

## Compile

`make` : normal mode

<br>

`make bonus` : Rotation matrix mode + alpha
- `arrow keys` move
- `r` reset
- `+` `-` zoom
- map size adjust
  - `a` `z` x axis adjust
  - `s` `x` y axis adjust
  - `d` `c` z axis adjust
- map rotation
  - `h` `n` x axis rotation
  - `j` `m` y axis rotation
  - `k` `,` z axis rotation

## Execute

`./fdf [map path]`
